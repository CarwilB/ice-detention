Facilities missing addresses

The facility roster should have an address and city for every facility. This document investigates the facilities that are currently missing one or both.

no_addr <- roster |>
  filter(is.na(facility_address) | facility_address == "" |
         is.na(facility_city) | facility_city == "") |>
  select(canonical_id, canonical_name, facility_city, facility_state,
         facility_type_wiki, detloc, id_range)

There are 41 facilities in the roster with no facility_address or facility_city.

Vera Institute coverage

The Vera Institute facility database contains 1,464 facility codes with addresses and geocoded coordinates. We can join by DETLOC to check whether Vera has address data for these missing facilities.

vera_check <- vera_facilities |>
  filter(detloc %in% no_addr$detloc) |>
  select(detloc, vera_name = facility_name,
         vera_address = facility_address, vera_city = facility_city,
         vera_state = facility_state, vera_zip = facility_zip)

no_addr_vera <- no_addr |>
  left_join(vera_check, by = "detloc")
vera_summary <- no_addr_vera |>
  summarise(
    total = n(),
    vera_has_address = sum(!is.na(vera_address)),
    vera_has_city = sum(!is.na(vera_city)),
    vera_has_neither = sum(is.na(vera_address) & is.na(vera_city))
  )
