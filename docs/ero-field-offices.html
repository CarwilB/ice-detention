ERO Field Offices as Informal Detention Sites

Published

March 22, 2026

Background

U.S. Immigration and Customs Enforcement (ICE) Enforcement and Removal Operations (ERO) maintains 25 field offices nationwide. These offices have always contained holding rooms — small, spartan spaces designed to hold people for a few hours after arrest, pending transfer to an official detention facility.

In June 2025, ICE issued an internal policy memo waiving its prior 12-hour limit on holds, extending permissible detention to up to three days. In practice, court records and data analyzed by the Guardian show the agency has routinely exceeded even this extended limit, holding people in unventilated concrete rooms for days and sometimes weeks — without the inspection, legal access, or oversight requirements that apply to official ICE detention centers.

This document harvests the 25 ERO field office listings from ICE.gov, builds a structured dataset, and compares those locations against the holding facilities named in the Guardian’s October 2025 investigation.

Scraping ERO Field Offices from ICE.gov

Approach and limitations

The ICE field offices page (https://www.ice.gov/contact/field-offices) lists offices of multiple types — ERO, HSI, OPLA, OPE, OPR — paginated 20 per page. The page accepts query parameters to filter by office type (?office=16 for ERO) and by page (?page=N, zero-indexed).

In principle this is scrapable with rvest. In practice, a few complications arise:

  1. Pagination: the full ERO listing spans two pages.
  2. Inconsistent title/name split: each listing has a short title line (e.g., "Burlington - ERO") and a longer official name (e.g., "Boston Field Office"). The city in the title is the physical address city; the official name sometimes refers to a different metropolitan area.
  3. Sparse fields: phone numbers and addresses are inconsistently structured across entries; two offices (Dallas, St. Rose) have no phone listed.

The scraping approach below fetches both pages, extracts the structured office blocks, and parses the title acronym as the office_type.

Fetching the paginated ERO listings

The ERO filter URL returns 20 results on page 0 and 5 on page 1 (25 total, matching ICE’s own statement that ERO “operates 25 field offices nationwide”).

library(xml2)
library(httr)
library(glue)

# 1. Correct the name and use the base URL
base_url <- "https://www.ice.gov/contact/field-offices"

fetch_ero_page <- function(page_num) {
  url <- glue("{base_url}?office=16&page={page_num}")
  
  # 2. Add a User-Agent header to look like a browser
  resp <- GET(
    url, 
    add_headers(`User-Agent` = "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebkit/537.36")
  )
  
  # 3. Check for successful status before parsing
  if (status_code(resp) != 200) {
    warning(glue("Page {page_num} failed with status {status_code(resp)}"))
    return(NULL)
  }
  
  Sys.sleep(2) # Being a bit more polite for government servers
  read_html(content(resp, "text"))
}

# 4. Use map but safely (optional: filter out NULLs)
pages <- map(1:2, fetch_ero_page) %>% compact()

Parsing office blocks

Each office on the page is rendered as a <div> with a title line and detail block. The title line follows the pattern "City - ACRONYM".

parse_ero_page <- function(html) {
  # Title lines are in <h2> or similar heading within each listing block;
  # detail text (address, phone, AOR) follows in paragraph elements.
  # NOTE: actual CSS selectors require inspection of the live DOM.
  # The extraction below is illustrative — adjust selectors as needed.

  blocks <- html |> html_elements(".views-row")

  map_dfr(blocks, function(b) {
    title       <- b |> html_element(".field-content h3, h2") |> html_text2()
    office_name <- b |> html_element(".views-field-title") |> html_text2()
    address_raw <- b |> html_element(".street-address, address") |> html_text2()
    phone       <- b |> html_elements("p") |> html_text2() |>
                        str_subset("^\\(\\d{3}\\)") |> first(default = NA_character_)
    aor_text    <- b |> html_elements("p, strong") |> html_text2() |>
                        str_subset("Area of Responsibility") |>
                        str_remove(".*Area of Responsibility:\\s*") |>
                        first(default = NA_character_)
    email_node  <- b |> html_element("a[href^='mailto']")
    email       <- if (!is.null(email_node)) html_attr(email_node, "href") |>
                        str_remove("^mailto:") else NA_character_

    tibble(
      listing_name  = trimws(title),
      office_type   = str_extract(title, "[A-Z]+$"),
      office_name   = trimws(office_name),
      address_raw   = trimws(address_raw),
      phone         = phone,
      aor           = aor_text,
      outreach_email = email
    )
  })
}

ero_scraped <- map_dfr(pages, parse_ero_page)

  • Santa Ana, CA is within the Los Angeles Field Office’s area of responsibility (which explicitly covers Orange County), but there is no dedicated ERO office listed there. This may be a sub-office or a non-ERO federal building.

The Guardian’s analysis further states that ICE has used “at least 170 ICE holding facilities nationwide, including at 25 ICE field offices.” That framing suggests the 25 ERO field offices represent a subset of a much larger informal detention infrastructure. The DDP data confirms this: it contains 173 distinct hold codes, including the 25 ERO field offices, 12 named sub-offices, and approximately 130 local hold rooms spanning nearly every U.S. state.

DDP DETLOC matching

The Deportation Data Project (DDP) daily population data contains 173 facility codes ending in “HOLD” — these represent holding rooms at ICE offices, sub-offices, and other locations. We match ERO field offices to their corresponding DDP hold codes.

ddp_codes <- tar_read(ddp_codes)

hold_codes <- ddp_codes |>
  filter(grepl("HOLD", detention_facility_code, ignore.case = TRUE) |
         grepl("HOLD|HOLDING", detention_facility, ignore.case = TRUE))

Canonical ID assignment

ERO field offices receive canonical IDs in the 2001–2025 range, reserved for ICE, CBP, POE, and other DHS facilities used for detention.

ero_canonical <- ero_field_offices |>
  left_join(ero_detloc_map |> select(office_name, detloc), by = "office_name") |>
  arrange(office_name) |>
  mutate(
    canonical_id = 2000L + row_number(),
    canonical_name = office_name,
    facility_type_wiki = "ICE ERO Field Office"
  ) |>
  select(canonical_id, canonical_name, address, city, state, zip,
         detloc, facility_type_wiki, area_of_responsibility,
